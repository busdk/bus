# Roadmap 0.0.1 — Core: Interfaces + Built-In Implementation Selection

## Goal
Establish Bus’s modularity model as internal Go interfaces with multiple built-in implementations selectable at runtime via config (no plugins).

## Progress
planned

- **Implementations**: built-in and selected deterministically.

None.

## API/CLI changes

## Automated testing goals
- Unit tests for deterministic selection behavior (unknown ids error deterministically).
